# FlaskApiTemplate

## Getting started

#### Running for the first time:

> ``` pip install pipenv ```
>
> ``` pipenv shell ```
>
> ``` pipenv install flask flask_restful ```
>
> ``` $env:FLASK_APP = "src/app.py" ```
>
> ``` $env:FLASK_ENV = "development" ```
>
> ``` flask run ```

*If you use Windows, run powershell as admin.*
